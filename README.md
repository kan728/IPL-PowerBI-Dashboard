# IPL Match Analysis Dashboard (Power BI)

This project analyzes Indian Premier League (IPL) match data using Microsoft Power BI.

## Dashboard Features
- Total IPL Matches
- Total Cities hosting matches
- Total Teams
- Matches by City
- Matches per Season
- Team Wins Analysis
- Toss Decision Distribution

## Insights
The dashboard highlights team performance across IPL seasons, match distribution across cities, and strategic decisions such as toss choices. It helps visualize trends in IPL match outcomes and team dominance.

## Tools Used
- Microsoft Power BI
- IPL Dataset (Matches & Ball-by-Ball)
